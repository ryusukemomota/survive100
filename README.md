# SURVIVE 100 - Battle Against Decay

健康意識を高めるための高速シミュレーションゲームです。20歳から100歳まで生き残ることを目指し、健康管理の重要性を体感できます。

![SURVIVE 100 Screenshot](https://via.placeholder.com/800x400/4F46E5/FFFFFF?text=SURVIVE+100+-+Battle+Against+Decay)

## 🎮 ゲーム概要

### 目標
20歳から100歳まで健康に生き残る（1プレイ10-15分）

### 特徴
- **高速シミュレーション**: 1ターン=1年、スピーディーな展開
- **戦略的リソース管理**: 年間4アクションポイント制
- **現実的な健康システム**: 糖化・炎症・変異レベルによる病気発症
- **教育的価値**: 予防投資の重要性を体感
- **周回プレイ**: 死因分析で次の人生戦略を改善

## 🏥 健康システム

### 管理するパラメータ
- **💰 ヘルス通貨（HC）**: 生活費・医療費
- **🛡️ 免疫力**: 感染症・がん耐性
- **💪 筋力**: 代謝・転倒防止
- **🧠 精神力**: ストレス耐性

### 病気システム
- **糖尿病**: 慢性疾患、症状管理のみ
- **がん**: 70%治癒率、高額治療費
- **自己免疫疾患**: 30%寛解率、免疫抑制リスク
- **感染症**: 90%治癒率、比較的軽症

## 🎯 ゲームルール

### 基本システム
1. **年間サイクル**: 毎年4APを使って行動選択
2. **収入システム**: 仕事でHC獲得、年齢で収入変化
3. **老化システム**: 年齢とともに最大ステータス減少
4. **病気発症**: リスクパラメータに応じて確率的に発生

### 戦略のポイント
- **予防 > 治療**: 病気になってから治すより予防投資が効率的
- **バランス**: 仕事（収入）と健康投資のトレードオフ
- **早期発見**: 健康診断による病気の早期発見
- **長期視点**: 若い頃の投資が老後の健康を決める

### 勝利・敗北条件
- **勝利**: 100歳まで生存
- **敗北**: 免疫力・筋力・精神力・資産のいずれかが0
